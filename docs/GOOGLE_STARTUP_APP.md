## 3. Technology Stack & Google Cloud Usage

### Built on Google Cloud ADK & Vertex AI

Axiom RESET is not just a wrapper; it is an **infrastructure project** built on our proprietary **AIX (Agent Identity Exchange)** standard.

#### Core Architecture

| Layer | Google Service | Usage |
|:------|:---------------|:------|
| **Intelligence** | Vertex AI (Gemini 2.0 Flash) | Multimodal reasoning for all agents |
| **Voice** | Cloud Speech-to-Text | Egyptian Arabic voice support |
| **Vision** | Vision AI + MedLM | Prescription OCR for Dr. Moe |
| **Documents** | Document AI | Contract generation for Tajer/Aqar |
| **Routing** | Google Maps Platform | Route optimization for Sofra/Tirs |
| **Compute** | Cloud Run | MCP server deployment |
| **Real-time** | Firebase | Voice command sync ("Arc Reactor") |
| **Analytics** | BigQuery | Market analytics for merchants |

#### The AIX Standard

We created a proprietary open standard called **AIX (Agent Identity Exchange)** that packages:

- **Soul:** Agent persona and ethical guidelines
- **Capabilities:** Connected MCP servers with latency SLAs
- **Economics:** Solana wallet for autonomous transactions
